# Monthly Expense Tracker

A single-file, mobile-first expense tracking web app built with vanilla HTML, CSS, and JavaScript.

## Features

- Track monthly income and expenses by category
- Month-by-month navigation with trend indicators
- Summary stats: total spent, remaining budget, savings rate
- Add, edit, and delete expense entries
- Glassmorphism UI with a branded gradient hero section
- No dependencies, no build step — open the HTML file in any browser

## Usage

Open `monthly_expense_tracker.html` directly in a browser. All data is stored in `localStorage` so it persists between sessions on the same device.
